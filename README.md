== Description ==

A WordPress plugin for subscription box businesses managing curated product selections and member shipments. Features customizable subscription tiers with skip and pause options, product curation tools with inventory reservation, shipment queue management with carrier integration, member portal for preference updates and tracking links, churn prediction alerts for at-risk subscribers, and monthly cohort retention reporting for business owners.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Subscription Box Curation and Member Management WordPress Plugin
